# Team headshots

Photos for the **Team** slide of the presentation decks. Square-ish headshots
work best (they're rendered in a 56×56 circle, `object-cover`).

Expected files (referenced by `VintigaOverviewSlidesScreen.tsx`):

| File | Person | Role |
|------|--------|------|
| `jim-secord.jpg`   | Jim Secord   | Founder |
| `donna-ataman.jpg` | Donna Ataman | Product & Customer Lead |
| `geoff-spears.jpg` | Geoff Spears | Engineering Lead |

If a file is missing the slide falls back to an indigo initials avatar, so the
deck never shows a broken image. Drop the files in with these exact names and
they appear automatically.
