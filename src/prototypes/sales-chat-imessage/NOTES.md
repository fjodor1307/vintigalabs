# Sales Chat — Notes

> Designer scratchpad. Open questions, things to revisit, observations.

- **Session-window UX:** we show a soft banner above the composer with time remaining ("Customer service window — 4h 23m left"). When expired, the banner turns into the "Only templates can be sent" hint. Is that the right place, or should it be inline with the last customer message? TBD.
- **Template variables UI:** currently a tiny stacked form. For real, we'd want preview-as-you-type so the agent sees the final rendered message.
- **Status ticks:** colour for read receipts — using `vintiga-indigo-500` for "read" feels on-brand but breaks WhatsApp muscle memory (blue). Worth asking real merchants.
- **Channel hint:** every conversation in the prototype is WhatsApp. Once we add SMS / Instagram, the inbox needs a channel column or per-row badge.
