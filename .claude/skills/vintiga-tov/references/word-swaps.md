# Word swaps

Replace the left column with the right. Sourced from the Vintiga voice on [vintigalabs.com](https://www.vintigalabs.com/) and the four principles in `SKILL.md`.

## Filler and politeness padding

| Avoid | Use instead |
|---|---|
| Please ensure you… | Make sure… |
| Kindly… | _(delete — just give the instruction)_ |
| In order to | To |
| At this time | Now |
| Going forward | From now on |
| Please note that | _(delete)_ |
| We would like to inform you | _(delete — say the thing)_ |

## Vague verbs

| Avoid | Use instead |
|---|---|
| Submit | Send / Book / Continue / Save _(match the next action)_ |
| OK | Got it / Continue |
| Click here | _(use a verb that names the action: "View report")_ |
| Manage | Edit / Add / Remove _(be specific)_ |
| Process | _(name the actual action)_ |

## Error and empty state language

| Avoid | Use instead |
|---|---|
| An error occurred. | Something went wrong. Try again. |
| Operation failed. | We couldn't save your changes. Try again. |
| Invalid input. | _(name the field and the rule: "Email needs an @")_ |
| No data available. | Nothing here yet — _(then say how to add the first one)_ |
| You are not authorised. | You don't have access to this. Ask your manager. |

## Corporate / marketing fluff

| Avoid | Use instead |
|---|---|
| Leverage | Use |
| Utilise | Use |
| Drive value / drive growth | _(say the specific outcome: "Bring guests back")_ |
| Streamline | _(name the friction you remove)_ |
| Best-in-class, world-class | _(delete — show it, don't claim it)_ |
| Cutting-edge, next-gen | _(delete)_ |
| Empower your team | _(name what the team can now do)_ |
| Solutions | _(name the product: "Mobile POS", "Wine Club")_ |

## Hospitality-specific

| Avoid | Use instead |
|---|---|
| User / customer | Guest _(in front-of-house contexts)_ |
| Staff member | Server / host / sommelier / your team |
| Transaction | Visit / order / check |
| Customer database | Guest profiles |

## Capitalisation reminders

- Buttons, titles, headers: **sentence case** ("Book a demo", not "Book A Demo").
- Feature names keep caps: **Mobile POS**, **Wine Club**, **CRM**, **Reservations**.
- No ALL CAPS for emphasis. Use weight or colour instead.
